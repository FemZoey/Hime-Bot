# Hime-Bot
>A discord bot written in typescript which features it's own built in api!

## Requirements:

- Discord.js 12.4.1 
`npm i discord.js@12.4.1`

- Moment 
`npm i moment`

- Quick.db
`npm i quick.db`

- Nekos.life
`npm i nekos.life`

- Request
`npm i request`

## Running:

After you've installed all of the packages in the requirements go to the folder and do:
`npm i main.ts`
